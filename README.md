# VanGo

VanGo uses genetic algorithms to select, mate, and mutate possible solutions to the given problem.

## Heuristics
- Fuzzy String Search
- Hamming Weight

### The Starry Night by VanGo
60 hours to recreate Vincent van Gogh's The Starry Night.
